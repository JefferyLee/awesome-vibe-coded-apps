# Contributing

This list is generated from [One's Vibe](https://onesvibe.app) — a curated
gallery of AI-built products that are live and testable. The README is
regenerated automatically, so pull requests editing it directly will be
overwritten.

## Want your app on the list?

[Submit it on One's Vibe](https://onesvibe.app/submit). Every submission goes
through an automated liveness check and human editorial review. Once
published, your app appears here in the next regeneration — and stays as long
as it stays alive (every link is re-checked on a rolling ~3-day cycle).

## Found a problem?

- **Dead or hijacked link** — it will drop off automatically within days, but
  you can speed it up by reporting the project on its One's Vibe detail page.
- **Wrong description or category** — report it on the project's detail page,
  or if it's your own project, claim it and request an edit.
- **Issues with the generator or list format** — open an issue here.
